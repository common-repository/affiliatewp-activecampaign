Easily add ActiveCampaign integration to AffiliateWP

== Description ==

The ActiveCampaign extension for AffiliateWP adds an opt-in checkbox to the affiliate signup page, enabling your affiliates to subscribe to your ActiveCampaign-powered newsletter.

Site admins can also select whether to show the checkbox, or simply autoregister all new affiliates.

== Installation ==

1. Activate the plugin
2. Navigate to Affiliates->Settings->Integrations
3. Enter your API information and select a list
